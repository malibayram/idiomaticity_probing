\section{Figures in Template}

\hspace{0.5cm} Figures must be centered, and captions set below the image. Image width must not exceed text width. Any text in images must not be larger than the main body text of the paper. An illustration of this guideline is presented in Figure \ref{fig:figure1}. Figures should be placed at the top or bottom of pages, and as close as possible to the text which references them. Authors should avoid placing figures in the first page.

\begin{figure}[!h]
    \centering
    \includegraphics[width=0.5\linewidth]{figures/figure_example}
    \caption{Example figure of a digital circuit.}
    \label{fig:figure1}
\end{figure}

We prefer vector image formats such as EPS. If EPS is not available, use high-quality images (minimum 200 dpi). Please note that figure images must be provided separately, alongside Latex and MS Word source files, when a paper has been accepted, for final editing.

\section{Tables in Template}

\hspace{0.5cm} Tables must be centered and captions set above the table, as presented in Table \ref{tab:table1}. Text should be centered in a cell, both vertically and horizontally. Tables, like figures, should be placed at the top or bottom of pages, and as close as possible to the text which references them. Authors should avoid placing tables on the first page.

\section{Equations}

\hspace{0.5cm} The following example displays an equation:

\begin{equation} \label{eq1}
\omega_T \times l_e +r_e \cong R_s.
\end{equation}

Equations are numbered and referenced in text as (\ref{eq1}), (\ref{eq2}), etc. Equations are inserted within the text, i.e., in phrases:

\begin{equation} \label{eq2}
\omega_o^2(l_b+l_e) \times (C_{be}+C_{bc})\cong 1,
\omega(l_b+l_e) = \frac{1}{\omega(C_{be}+C_{bc})},
\end{equation}

\noindent so please use punctuation as illustrated in (\ref{eq1}) and (\ref{eq2}).

\section{References}

\hspace{0.5cm} References should be cited in text in IEEE style as follows: \cite{1}, \cite{2}, \cite{3--6}. The References section must list all cited references in the order of citation, i.e., the reference cited first is listed first and so on.

You should place bracketed citations within the line of text, before any punctuation, with a space before the first bracket.\\
