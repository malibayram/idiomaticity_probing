The official ROMJIST LaTeX template was downloaded via Chrome to your Downloads folder
as "Template_Latex.zip" (from https://www.romjist.ro/Template_Latex.zip).

To finish setting up:
  1. Move Template_Latex.zip from ~/Downloads into this folder.
  2. Unzip it here (double-click, or: unzip Template_Latex.zip).
  3. In ../main.tex, switch \documentclass to the ROMJIST class file (.cls) from the template.

Also download and keep alongside the final paper:
  - Copyright form:  https://www.romjist.ro/Copyright_form_ROMJIST_v2.pdf
  - Cover letter:    https://www.romjist.ro/cover_letter_template.doc
